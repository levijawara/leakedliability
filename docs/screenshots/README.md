# Screenshots Guide

Add product screenshots here and keep filenames stable so README links do not break.

## Recommended files

- `leaderboard.png`
- `submit-report.png`
- `admin-dashboard.png`
- `escrow-flow.png`

## Capture guidelines

- Use realistic but non-sensitive sample data.
- Redact personal emails, phone numbers, and payment identifiers.
- Prefer 16:9 or 4:3 captures at readable resolution.
- Keep UI in a representative production-like state.

## Suggested alt-text style

- "Leaderboard view showing producer rankings and payment reliability indicators."
- "Submit Report flow with timeline and payment detail fields."
- "Admin dashboard with moderation and analytics controls."
